Welcome to Zapata's documentation!
==================================

This containd the assembled library and codes jointly developed at CMCC.
It contains mapping, computational and utilities for working with CMCC
Supercomputing Center, accessing its data and storage systems.

Access credentials can be obtained from Osvaldo.

The codes are also in the CMCC GitHub Repository CMCC-Foundation/Applications
and they should be dowonloaded and installed to your local machine.

Installation instructions are here.

Happy Computing !
