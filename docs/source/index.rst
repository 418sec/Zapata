Welcome to CMCC Data Analysis Library !
=======================================

This is the assembled library and codes jointly developed at CMCC.
It contains mapping, computational and utilities for working with CMCC
Supercomputing Center, accessing both data on storage systems and computational resources.

Access credentials for CMCC SCC facility can be obtained iupon request to hsm@cmcc.it.

The source of this library is available at CMCC GitHub repository https://github.com/CMCC-Foundation/Zapata

Installation instructions are given in the README.md file of the package.

Checkout the available Jupyterlab Notebooks within the examples_notebook folder to see the potential applications of the library.
